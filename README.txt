Project Title
This project is about the movie trailer website, which allows the users to watch their favourite movie trailers.


Getting Started
Follow the instructions to get watch the exiciting trailers.

Prerequisites
A windows/mac/linux machine to install "python ide".

For windows machine-
visit python.org and download the version compitable to your system.
Next step is to install python ide to your machine.

after the succesfull installation, open the file "entertainment.py" and run it by simply pressing F5 or from the run menu.


After giving command to run, the default browser will open a tab i.e.Box Office.
The Box Office website will have the movies trailers of the users choice. 
And users can watch the trailers by clicking on their favourite movie.
